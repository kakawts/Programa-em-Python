# Programa-em-Python
Programa em Pyhton para avaliar desempenho de uma empresa com três filiais.

Desenvolvi um programa simples em Python que permite avaliar o desempenho de uma empresa que possui três filiis, analisando o faturamento e o fluxo de clientes de cada filial ao longo de quatro semanas.

O programa conta com um menu de opções que permite Cadastrar Dados, Gerar Relatório de Desempenho, Atualizar Dados, Gerar Ranking de Faturamento e Sair para encerrar o programa.

Além disso o programa conta com processos de validação para evitar a inserção de valores negativos, contribuindo para o funcionamento correto e evitando erros.
